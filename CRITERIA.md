# 블로그 소재 분류 기준

## 내 프로필
- 아이 **30개월** (영아/유아 경험 O, 초등학생 경험 X)
- 직업: 개발자
- 관심사: 육아(영어/교육), 재테크, 부동산, 블로그/수익화, 창업, 개발/IT

---

## can_i_solve 기준 (내가 직접 쓸 수 있는 주제)

### ✅ 가능한 카테고리

| 카테고리 | 조건 |
|---|---|
| **육아** | 0~4세 영아/유아 관련, 영어 교육, 어린이 교육법 위주 |
| **부동산** | 전월세, 매매, 계약, 인테리어, 집수리 전반 가능 |
| **재테크** | 주식, ETF, 보험, 절세, 저축 등 |
| **교육** | 초·중·고·대학교 진학, 과목 선택, 학원/자격증 전략 |
| **개발자/IT** | 개발, 기기 설정, 자동화, AI 도구 활용 |
| **블로그/수익화** | 블로그, 유튜브, 쇼츠, 네이버 클립, 콘텐츠 전략 |
| **창업** | 창업 아이템 탐색, 초기 전략, 폐업 경험 — 나도 배우는 입장 |
| **업무/생산성** | 실행력, 루틴, 시간관리 |
| **영어/이민** | 영어 학습, 해외 생활 정보 |

### ❌ 제외 기준

| 제외 이유 | 예시 |
|---|---|
| 초등학생 사회적 관계 (경험 없음) | 초등 친구관계, 교우문제, 왕따 |
| 의료/법률 전문 영역 | 수술 결정, 법적 분쟁 |
| 순수 광고/스팸 | 중고차, 법률광고, 제품 홍보 |

## is_blog_material 기준

**True 조건** — 다음 중 하나 이상:
- 개인 고민/질문 (막막함, 헷갈림, 실패 경험 포함)
- 정보 갭: "아무도 안 알려주는", "이거 몰랐어요"
- 시행착오/삽질 경험 공유
- 의사결정이 필요한 상황 (A vs B 비교)

**False 조건** — 다음 중 하나라도 해당:
- 광고/스팸/법률광고/중고차 광고
- AI 생성 글 (제목이 너무 매끄럽거나 내용 없음)
- 단순 일기/감상/에세이
- 제품 리뷰, 후기 나열
- 정치/연예/스포츠

---

## 출력 컬럼 설명

| 컬럼 | 설명 |
|---|---|
| `is_blog_material` | 블로그 소재로 쓸 만한 글인지 (True/False) |
| `category` | 글의 주제 카테고리 |
| `can_i_solve` | 내가 직접 답/블로그를 쓸 수 있는지 (True/False) |
| `blog_angle` | 이 글을 블로그로 쓴다면 어떤 각도로 쓸지 |
| `desc_summary` | 원문 description 앞 100자 요약 |
